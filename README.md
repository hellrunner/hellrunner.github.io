## 🔥 О проекте

**HLRN_pass_gen** — генератор паролей из ада. Работает в браузере и из консоли.
Случайность — через криптостойкий генератор (**Web Crypto API** / **`secrets`**).
Данные никуда не уходят — всё рождается локально.

## 🌐 Веб-версия

> [**hellrunner.github.io/pass_gui**](https://hellrunner.github.io/pass_gui/)

| Режим | Описание |
|-------|----------|
| **hex 16** | Аналог `openssl rand -hex 16` — 16 байт, 32 hex-символа, **128 бит** энтропии |
| **Обычный пароль** | Настраиваемая длина, заглавные буквы, цифры, спецсимволы |

---

## 💻 Консольная версия

```bash
python password_generator.py
```

Программа запросит длину пароля (не менее **4** символов) и выдаст надёжный пароль:
строчные и заглавные буквы, цифры, спецсимволы.
